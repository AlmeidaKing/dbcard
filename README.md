# DB CARD
## DB CARD - API

Aplicação backend em NodeJS utilizando Typescript

## Pré-requisitos
- NodeJS
- MongoDB

## Instalação

Baixar o repositório e executar 
`
yarn
`
## Utilização

A aplicação escuta requisições nas rotas configuradas para executar ações CRUD nos repositórios.
